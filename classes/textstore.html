Class TextStore

Super performant text container built upon caches

Quick Notes: When interacting using ITextPositions, keep in mind that lowest value is { line: 1, col: 1 } When using indices, lowest value is 0.

{ line: 1, col: 1 } === Index 0

Also remember that most operations that accept ITextPosition, ITextRange and indices, require them to be within document bounds. Only exception is TextStore.prototype.getContents which auto-clamps the range
