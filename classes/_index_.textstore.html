Class TextStore

Super performant text container built upon caches

When interacting using TextPositions, keep in mind that lowest value is { line: 1, col: 1 } When using indices, lowest value is 0

Also remember that most operations accepting TextPosition, TextRange and indices require them to be within document bounds. Only exception is TextStore.prototype.getContents which auto-clamps the range
